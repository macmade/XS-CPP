XS::Threading::STLMutex::STLMutex ( bool  recursive = true)

STLMutex Class constructor

Parameters
recursiveWhether to create a recursive mutex or not If the mutex if created as recursive (default), it may be successfully locked multiple times by the same thread. If the mutex isn't recursive, a dead-lock will occur if it is locked twice from the same thread. Note that if a recursive mutex is locked multiple times by a thread, each call to the 'Lock' method shall be balanced by a call to the 'Unlock' method.
