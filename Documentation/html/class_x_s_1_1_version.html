std::string XS::Version::ToString ( void  ) const
virtual

Gets a string representation of the version.

Returns
A string representation of the version

This method will return a string representation in the following form: [major].[minor].[build]-[status][revision] eg: 1.2.3-rc4 Note that for version objects with a "final" status, the status/revision part will be ommited if the revision number is zero.
